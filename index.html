HTML & CSS Cheat Sheet

This page will show how to setup an HTML Table

HTML Table

Tag Name Description
<table> Table The wrapper element that wraps all HTML Tables
<thead> Table Head The set of rows defining the column headers
<tbody> Table Body The set of rows actually containing table data
<tr> Table Row element that defines a table row
<td> Table Data The container for table cell data
<tfoot> Table Footer The set of rows defining a table footer

Table Attributes

Attribute Name Description
<colspan> Column Span Defines how many columns a td element should span
<rowspan> Row Span Defines how many rows a td element should span

Raw Table Code

<table>
  <thead>
    <tr>
      <td>Attribute</td>
      <td>Name</td>
      <td>Description</td>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><colspan></td>
      <td>Column Span</td>
      <td>Defines how many columns a td element should span</td>
    </tr>
    <tr>
      <td><rowspan></td>
      <td>Row Span</td>
      <td>Defines how many rows a td element should span</td>
    </tr>
  </tbody>
</table>
